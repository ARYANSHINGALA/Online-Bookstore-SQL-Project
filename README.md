Online Bookstore SQL Project
Project Overview

This project demonstrates SQL skills by analyzing an Online Bookstore database using three related datasets.


The project includes beginner to advanced SQL queries covering:

  SELECT | WHERE | ORDER BY | GROUP BY | Aggregate Functions | JOINS | Subqueries | Data Analysis |Database.\
  

Three CSV files were used.

Table	: Description => Books : Book information | Customers :	Customer details | Orders :	Customer orders
